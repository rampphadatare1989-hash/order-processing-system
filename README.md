# Order Processing System

This repository contains the source code for the Order Processing System, an Angular and Firebase-based application for managing orders, products, and job cards.

## Features
- Dashboard with progress visualization
- Product image upload (max 6 images)
- Job card detail and report with product images
- Print-friendly layouts
- Firebase integration

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm start` to launch the development server

## Deployment
Deployed on Firebase Hosting.
